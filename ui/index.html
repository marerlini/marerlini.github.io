UI controls


A radio button is a user interface control that lets a user select only one option from a set of mutually exclusive choices.
A checkbox in a user interface is a small, interactive square that allows users to select one or more options from a list, or to indicate a binary choice.
Text input in a user interface refers to a component that allows users to enter textual information into a system or application.
